# Free Kick Master

Браузерна аркадна футбольна гра, де гравець виконує серії штрафних ударів, проходячи рівні з поступовим ускладненням.

## Опис гри

"Free Kick Master" — це аркадна футбольна гра, у якій гравець виконує серії штрафних ударів, проходячи рівні з поступовим ускладненням умов. На кожному рівні гравець виконує 5 ударів. За результатами цих ударів він отримує внутрішньоігрову валюту — монетки. Монети використовуються для покупки нової форми для футболіста та різних типів м'ячів.

## Система винагород

- **Гол:** 10 монет
- **Влучання в мішень:** 5 монет (якщо мішень присутня)
- **Проходження рівня:** бонус 5 монет

Монети можна витрачати в магазині на:
- Форму (різні комплекти: базові, рідкісні, преміум)
- М'ячі (класичний, золотий, важкий, легкий тощо)

## Технології

- **JavaScript (ES2022)** - основна мова розробки
- **Vite** - build tool та dev server
- **Three.js** - 3D рендеринг
- **Web Audio API** - звукові ефекти та музика
- **localStorage** - збереження прогресу

## Як запустити локально

### Вимоги

- Node.js 18+ 
- npm або yarn

### Встановлення

```bash
# Встановити залежності
npm install
```

### Запуск

```bash
# Запустити dev server
npm run dev
```

Гра відкриється автоматично в браузері на `http://localhost:3000`

### Збірка для production

```bash
# Створити production build
npm run build

# Переглянути production build
npm run preview
```
